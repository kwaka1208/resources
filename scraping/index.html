スクレイピングしてみよう

  • 商品一覧
    1. 商品一覧から商品名を取得して表示してください
    2. 取得した商品名をcsv形式にして表示してください 
            りんご,バナナ,オレンジ,ぶどう
  • ニュース見出し
    1. ニュースの見出しを取得して表示してください
    2. 取得した見出しを以下の形式のJSONにして表示してください（改行はこの通りでなくても構いません） 
            {
          "headlines": [
            "Python入門：はじめの一歩",
            "スクレイピングとは何か？",
            "AIと未来の仕事"
          ]
        }
  • 天気予報
    1. 天気予報、都市名、天気、気温の情報を取得して、ひとつの都市を一行で表示してください
    2. 取得した天気予報情報を以下の形式のJSONにして表示してください（改行はこの通りでなくても構いません） 
            {
          "weather": [
            {"city": "東京", "condition": "晴れ", "temperature": "25℃"},
            {"city": "大阪", "condition": "くもり", "temperature": "22℃"},
            {"city": "札幌", "condition": "雨", "temperature": "18℃"}
          ]
        }
